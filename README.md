# DDC-project | Marwiyyat

Marwiyyat is an interactive geographic platform that allows users to explore historical Islamic locations through an interactive map, multilingual stories, audio narration, and location-based discovery.

## Features

- Interactive map visualization
- Nearest place detection using PostGIS
- Arabic and English support
- Audio storytelling
- Story summaries
- Location images
- Story view counter

## Tech Stack

- React
- Node.js
- Express.js
- Supabase
- PostgreSQL
- PostGIS
- Cloudinary
- Leaflet

## Database

The system uses two main tables:

- places
- place_translations

The multilingual content is managed through the place_translations table, allowing Arabic and English versions of stories, summaries, audio files, and metadata.
